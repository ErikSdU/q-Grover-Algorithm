  • Descubriendo el Algoritmo de Grove

    • Se utiliza para la busqueda de secuencias no ordenadas,basicamente nos ayuda a encontrar valores poco frecuentes.
    Al ser un algoritmo cuantico busca los datos de una manera mas eficiente que los algoritmos tradicionales

  • Aplicaciones

    • Aunque el propósito del algoritmo es la búsqueda en una secuencia se puede encontrar de una manera más adecuada como la "inversión de una función". Así, teniendo la función y=f (x), que puede ser evaluada en un computador cuántico, con este algoritmo calculamos el valor de x cuando se nos da como entrada el valor de y. Invertir una función puede relacionarse con la búsqueda en una secuencia, si consideramos que la misma es una función que produce el valor de y como la posición ocupada por el valor x en dicha secuencia.
